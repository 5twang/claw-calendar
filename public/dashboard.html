快速开始

1

获取 API Key

前往 API Keys 页面 创建访问密钥，复制生成的密钥。

2

配置 WorkBuddy

推荐 让 WorkBuddy 自动完成

只需把 API Key 告诉 WorkBuddy，它会自动保存并配置好。

我：
"这是我的 API Key: claw-calendar-xxxxxxxxxxxxxxxx"
✅ 已创建「默认」日历
✅ 配置已保存
📅 默认日历: 默认
💡 手动配置（备用）

在 Calendar Skill 目录下创建 .env 文件：

# Claw Calendar API 配置
CALENDAR_API_URL=https://api.claw-calendar.com
CALENDAR_API_KEY=你的API密钥
3

用 WorkBuddy 创建日程

告诉 WorkBuddy 你想创建什么日程，例如：

我：
"明天下午3点有一个产品评审会，帮我添加到日历"
WorkBuddy：
"已为你创建日程：产品评审会
时间：2026-04-14 15:00
是否需要提醒？"

常用场景示例

📅 添加生日提醒
"4月20日是妈妈的生日，提前一天提醒我"
💼 安排会议
"下周一上午10点预约李明开会讨论项目进度"
📚 制定计划
"从明天开始每天早上8点提醒我背单词"
